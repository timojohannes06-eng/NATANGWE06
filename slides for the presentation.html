The Problem

Corrosion Costs Lives & Money

🏗️

Manual & Inconsistent

Traditional inspections rely on visual assessments by engineers — subjective and time-consuming.

📋

Poor Documentation

Records are often incomplete, delayed, or lost in the field — no standardised system exists in Namibia.

⏳

No Trend Tracking

Without systematic records, it is impossible to monitor corrosion progression over time and act early.

CorroCheck solves this by putting a structured, AI-assisted inspection tool directly in every inspector's pocket.

Technology

The Stack

📱 Mobile Framework

React Native + Expo (Managed Workflow)

Android-first, targets API Level 29+. Expo Go used during dev; EAS Build for final APK.

🔐 Authentication

Firebase Authentication

Email & password registration and login. Unique UID per user ties all data together.

🗄️ Database

Firebase Firestore

NoSQL cloud database. Stores inspection reports in a users/{userId}/scans sub-collection.

🖼️ File Storage

Firebase Storage

Stores uploaded inspection photos. Returns a URL saved alongside each scan record.

🤖 AI Analysis

Roboflow API + Node.js

Node.js server securely forwards photos to Roboflow. Returns severity + confidence score.

🎨 Design

Figma + Inter font

Blue & gray palette. Designed on iPhone 15 Pro frame. Full interactive prototype delivered.

Firebase Architecture

Firestore Data Model

📁 users (collection)

userId — String (Firebase UID)
name — String
email — String
createdAt — Timestamp

📁 users/{userId}/scans (sub-collection)

scanId — Auto-ID
photoURL — String (Firebase Storage link)
corrosionType — Uniform / Galvanic / Pitting / Crevice
severity — Low / Medium / High (Roboflow)
location — GPS coords or manual text
date — Timestamp
notes — String (optional)
confidence — Number (AI %)

Security rule: request.auth != null required for all reads and writes.

Core Feature

AI Analysis Flow

📷 User captures photoin app
→
📤 App sends phototo Node.js server
→
🔁 Server forwardsto Roboflow API
→
🤖 Roboflow analysesseverity + confidence
→
📨 Result returnedto app
→
💾 Saved to Firestore+ shown to user
LOW
Minimal rust, surface discoloration
MEDIUM
Active rust, structural risk warning
HIGH
Severe degradation, immediate attention

Node.js keeps the Roboflow API key secure — never exposed in the mobile app.
